# VouEx — Voucher Exchange Platform

A web platform for discovering, sharing, and redeeming discount vouchers and coupon codes.

## Features

- Browse vouchers by category
- View detailed coupon information and redemption instructions
- User registration and login system
- Coupon code reveal and copy functionality
- Clean, responsive UI

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Homepage with featured vouchers |
| `coupon.html` | Full voucher listing |
| `coupon-detail.html` | Individual voucher details |
| `coupon-code.html` | Reveal & copy coupon code |
| `login.html` / `register.html` | User authentication |
| `about_us.html` | About the platform |
